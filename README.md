# Paper Search Agent

This is boilerplate code for a customer feedback analysis agent.

## Setup

Install the Python dependencies.

```bash
pip install -r requirements.txt
```

## Run the agent

```bash
python -m papersearch_agent.agent.papersearch_agent
```

## Requirements

- Python 3.10+
- autogen
- mistralai
- ollama
- fix-busted-json
- semanticscholar
